Display beautiful custom badges for WooCommerce product attributes with custom colors and icons.

== Description ==

Codenitive Badges for WooCommerce allows store owners to display product attribute terms as stylish badges on shop and product pages.

Each attribute term can have:

* Custom background color
* Custom text color
* Custom icon
* Automatic badge rendering
* Global badge styling options

Perfect for highlighting:

* Strain Types
* Product Types
* Strength Levels
* Flavors
* Categories
* Custom Product Attributes

= Features =

* Display WooCommerce attribute terms as badges
* Custom colors per attribute term
* Optional icons for badges
* Global badge style settings
* Works on Shop and Product pages
* Lightweight and WooCommerce-friendly
* No theme modifications required

== Installation ==

= Automatic Installation =

1. Upload the plugin through Plugins → Add New → Upload Plugin.
2. Activate the plugin.
3. Ensure WooCommerce is installed and activated.

= Manual Installation =

1. Upload the plugin folder to:

/wp-content/plugins/

2. Activate the plugin from the WordPress Plugins screen.

== Usage ==

= Configure Global Badge Settings =

Navigate to:

WooCommerce → Badge Settings

Configure:

* Default Background Color
* Default Text Color
* Badge Border Radius
* Badge Font Size

Save settings.

= Configure Attribute Term Badges =

Go to:

Products → Attributes

Edit an attribute term.

You will see additional fields:

* Badge Background Color
* Badge Text Color
* Badge Icon

Save the term.

= Display Badges =

Badges automatically appear on:

* Shop pages
* Category pages
* Product pages

No shortcode required.

== Frequently Asked Questions ==

= Does this require WooCommerce? =

Yes. WooCommerce must be installed and activated.

= Can I use multiple attributes? =

Yes. The plugin supports all WooCommerce global attributes.

= Does it work with variable products? =

Yes. Badges can be displayed for attributes assigned to variable products.

= Can I use custom icons? =

Yes. You can enter emoji icons or icon classes depending on your implementation.

= Does it affect WooCommerce performance? =

No. The plugin is lightweight and only loads the necessary badge data.
